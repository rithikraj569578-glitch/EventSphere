# EventSphere
ventSphere is a modern event discovery and management platform built with React, HTML, CSS, and JavaScript. Users can browse events, book tickets, generate QR passes, manage wishlists, and get AI-powered recommendations. It also features an organizer dashboard for analytics, attendee check-ins, ticket sales, and event management.
